# Dnyansampada Library Web Portal

A premium, modern, nature-themed web portal for **Dnyansampada Public Library (ज्ञानसंपदा सार्वजनिक वाचनालय ग्रंथालय)**, Nashik.

Developed by **[Ganesh Jamadar](https://github.com/GaneshJamadar369)**.

## 🌟 Features

- **Nature-Inspired Aesthetic:** Cohesive design using warm earth tones, forest greens, and a custom floating/falling leaves animation overlay.
- **Premium 3D Assets:** Custom-generated high-fidelity 3D assets for library features (Books, Community, Events).
- **Introductory Animations:** A smooth, immersive intro animation showing traditional Indian literature texts rising like wisdom.
- **Responsive Layout:** Beautiful typography, fluid grid structures, and full accessibility on all device sizes.
- **Modern Tech Stack:** React, Next.js (App Router, Turbopack), Tailwind CSS (v4), Framer Motion, and Lucide React.
- **Clean Structure:** Configured with Shadcn UI style base-nova for premium interface components.

## 🛠️ Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS (v4)
- **Animations:** Framer Motion
- **Icons:** Lucide Icons & Custom 3D Renders
- **Languages:** TypeScript / JavaScript

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the application:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the library portal.
